## D. 主应用体验打磨

### D1. 快捷键：Ctrl+K 全局搜索 + Ctrl+Shift+N 新建会话

- 文件：`src/agent_harness/static/index.html`
- `keydown` 监听器已存在（~line 87），新增两个条件分支
- Ctrl+K → focus sidebar search input
- Ctrl+Shift+N → 调用 `newChat()` 函数

### D2. 消息复制按钮

- 每个 assistant/user 消息渲染函数中，添加「📋」复制按钮
- 点击复制消息 Markdown 原文到剪贴板
- 按钮 hover 可见，disappear 其他时间

### D3. 代码块语言标签

- 渲染 Markdown 代码块时，`class="lang-xxx"` 会解析
- 在代码块左上角显示语言名称标签（如 `python`、`json`）
- 已有 `<pre><code>` 渲染，只需 CSS + 微调渲染逻辑

### D4. 搜索建议（自动补全）

- 搜索输入框监听 input 事件
- 从 localStorage 读取历史搜索词
- 显示下拉建议列表，点击/Enter 选中

## E. Agent 核心增强

### E1. 搜索链可靠性加固

- `tools/web.py` — 增加多页面解析 fallback
- 当前 SearXNG → DuckDuckGo → skill，DDG 只用一个 selector
- 新增第 2/3 selector 兜底 + 无头浏览器兜底（Playwright）

### E2. LLM 响应缓存

- `pipeline/llm.py` — 添加请求级缓存
- 基于 `(model, messages_json, temperature)` 的 MD5 hash
- 内存 dict + TTL 60s
- 仅缓存 GET-like 幂等调用

### E3. 报告引用格式优化

- 引用来源加入超链接锚点
- 报告摘要句子级引用标记（[1][2] 格式）
- 基于 `report_formatter.py` 已有模板
