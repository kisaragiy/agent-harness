# 灵枢 (LingShu) — 产品定义

> **灵枢者，智之枢也。** 以 Supervisor 为枢，Worker 为四肢，调度万端。
> 一个本地优先、开箱即用的多 Agent 调研助手平台。

---

## 一、产品定位

**一句话：灵枢是运行在你电脑上的 AI 调研助手。**

它不是聊天机器人（虽然你可以跟它聊天）。
它不是云服务（虽然它能用云 API）。
它不取代 ChatGPT（它做的是 ChatGPT 做不到的事——搜索→分析→出报告）。

灵枢的核心是 **Supervisor-Worker 架构**：
- 你告诉它"调研什么"
- 它自己决定"怎么搜、怎么分析"
- 做完了给你一份正式报告

## 三、它不应该做什么

| 不做 | 原因 |
|------|------|
| **不做云服务** | 数据隐私、网络依赖、延迟不可控 |
| **不做聊天机器人平替** | 灵枢的目标是出报告，不是聊天 |
| **不做全栈 IDE** | 灵枢写代码给你看，你自己决定用不用 |
| **不自动执行危险操作** | 桌面操作、代码执行、发消息都需显式确认 |
| **不绕开系统限制** | 不提权、不改注册表、不动系统文件 |
| **不做 CRM/工单/客服系统** | 不是这类产品的定位，面试需要可做独立 Demo |

## 五、技术栈

| 层 | 技术 |
|------|------|
| 窗口 | pywebview (WebView2) |
| 后端 | Python 3.11+ / FastAPI / Uvicorn |
| 编排 | LangGraph (Supervisor-Worker) |
| 认证 | HMAC-SHA256 JWT + pbkdf2_hmac 密码哈希 + SQLite |
| 向量 | Ollama nomic-embed-text → NPY 文件 → BM25 降级 |
| 搜索 | SearXNG → DuckDuckGo HTML → skill 三级降级 |
| 存储 | SQLite / JSON 文件 / NPY |
| 技能 | SkillHub CLI (+ 目录 moveto _disabled/) |
| 打包 | PyInstaller (单文件夹 exe, ~130MB) |
| 推理 | llama.cpp / Ollama / DeepSeek Flash API |

## 七、核心理念

> **灵枢不是一个更聪明的 AI，它是一个能帮你干活的 AI。**

- **搜索→分析→出报告，核心链路不偏**
- **能本地就不上云，能离线就不联网**
- **开箱即用 > 功能完备**
- **出错不要紧，告诉你怎么修**
- **版本慢一点，稳定第一**
