# 灵枢 (LingShu) V1.0 — 通用超级智能体平台

> **灵枢者，智之枢也。** 以 Supervisor 为枢，Worker 为四肢，调度万端。
> 一个本地优先、开箱即用的通用多 Agent 超级智能体平台。

---

## 一、产品定位

**一句话：灵枢是运行在你电脑上的通用 AI Agent 平台。**

它不是聊天机器人（虽然你可以跟它聊天）。
它不是云服务（虽然它能用云 API）。
它是一个通用的 Agent 底座——你可以用它做调研、搭客服、跑定时任务、管理知识库。

灵枢的核心是 **Supervisor-Worker 架构 + 插件扩展**：
- 你告诉它"做什么"
- 它自己决定"怎么做"  
- 插件系统让你扩展它的能力

---

## 二、它能做什么

### 已实现（v0.12.1）

| 能力 | 说明 | 使用方式 |
|------|------|---------|
| **多 Agent 编排** | Supervisor 分析任务 → 并行派发 Search/Analyze/Execute Worker | 对话中自然触发 |
| **网页搜索** | 通过 SearXNG 或 Bing 搜索实时信息 | 问需要搜索的问题 |
| **代码执行** | Python 代码沙箱，可运行任意 Python 代码 | "用Python算一下..." |
| **文件操作** | 读写工作区内文件 | "读取 README.md" |
| **知识库 RAG** | 上传 PDF/DOCX/TXT → 自动索引 → 问答检索 | 对话页下方上传 |
| **会话持久化** | 对话历史存磁盘，重启不丢 | 自动（X-Session-Id） |
| **桌面自动化** | 启动应用、模拟点击、截图OCR、窗口管理 | 需在设置中启用 |
| **A 股数据** | 实时行情、技术指标、财报数据 | "查一下茅台股票" |
| **ComfyUI 绘画** | AI 文生图/图生图/角色三视图 | 需启动 ComfyUI |
| **MCP 协议** | 40+ 工具通过 MCP 暴露，外部 Agent 可调用 | 查看 菜单 > MCP 列表 |

### 规划中

- 定时任务（cron 触发 Agent 干活）
- 插件系统（第三方注册工具）
- 自定义 Worker 模板

## 三、它不应该做什么

| 不做 | 原因 |
|------|------|
| **不做云服务** | 数据隐私、网络依赖、延迟不可控 |
| **不做聊天机器人平替** | 灵枢的目标是执行任务，不是聊天。你想聊天请用 ChatGPT |
| **不做全栈 IDE** | 不写代码？灵枢不是 Cursor。灵枢是写代码给你看，但你自己决定用不用 |
| **不自动执行危险操作** | 桌面操作、文件写入、发送消息都需要明确指令或权限确认 |
| **不绕开系统限制** | 不提权、不改注册表、不动系统文件。灵枢在你的用户权限内运行 |

## 五、技术栈

| 层 | 技术 |
|------|------|
| 窗口 | pywebview (WebView2) |
| 后端 | Python 3.11+ / FastAPI / Uvicorn |
| 编排 | LangGraph (Supervisor-Worker) |
| 向量 | Ollama nomic-embed-text + NPY + 关键词降级 |
| 存储 | JSON 文件 (会话/配置/RAG chunks) |
| 打包 | PyInstaller (单文件夹 exe, ~130MB) |
| 推理 | llama.cpp / Ollama / DeepSeek API |

## 七、核心理念

> **灵枢不是一个更聪明的 AI，它是一个能帮你干活的 AI。**

- **你说做什么，它决定怎么做**
- **能本地就不上云，能离线就不联网**
- **开箱即用 > 功能完备**
- **出错不要紧，告诉你怎么修**
