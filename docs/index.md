## 项目简介

**Agent Harness** 是一个面向 AI Agent 开发者的全流程框架，提供从原型验证到生产部署的一站式解决方案。无论你是构建对话机器人、自动化工作流，还是复杂的多 Agent 协作系统，Agent Harness 都能让你专注于业务逻辑，而非基础设施。

### 核心特性

| 特性 | 说明 |
|------|------|
| 🧩 **模块化架构** | 插件化工具系统，按需加载，灵活扩展 |
| 🚀 **一键启动** | 零配置即可运行，5 分钟上手构建第一个 Agent |
| 🔧 **强大的工具链** | 内置 80+ 工具：浏览器自动化、代码执行、文件操作等 |
| 📊 **内置评测** | 完整的自动化评测系统，量化 Agent 性能指标 |
| 🔐 **安全隔离** | 沙箱执行环境，保障系统安全 |
| 🐳 **容器化部署** | 原生 Docker 支持，一键部署到云端 |
| 📝 **CS Demo 模式** | 客户成功演示场景，快速展示产品价值 |
| 🌐 **REST API** | 完整的 HTTP API，方便集成到现有系统 |

### 快速体验

```bash
# 安装
pip install agent-harness

# 启动服务
agent-harness serve

# 浏览器打开 http://localhost:8000
```
